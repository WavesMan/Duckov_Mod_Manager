# Duckov Mod Manager 鸭科夫Mod管理器

这是一个游戏《逃离鸭科夫》的第三方Mod管理器工具，我们与《逃离鸭科夫》官方没有任何关系，请注意区别

## 关于 Duckov
《逃离鸭科夫》(Escape from Duckov) 是一款搜打撤生存射击游戏，支持丰富的Mod系统。

## 关于 Duckov Mod Manager

### Mods来源

当前完全依赖Steam创意工坊

### Mods管理

已支持游戏外管理Mods是否启用，不再为反复重启游戏而烦恼
